Before starting please fix the following

==.env==
- Token: Change this to your Discord bots token 
- Prefix: Change this to the deafult prefix of the bot (deafult .)
- Color: What color should the embeds be by default (in hex)

==/utils/moongoose.js==
Line 15: Change the following variables
- This was my default connect copy paste command please change this to your connect url
- <password> This should be set to the password of the databe account name
- <dbname> Change this to the database name

==npm init==
Open a command promt and type
npm init

==Install the npm packages==
npm install discord.js
npm install dotenv
npm install mongoose
npm install ascii-table
npm install cli-color
